My Hobbies

真正的休闲不是说什么都不做，而是自由地做他们感兴趣的事.

旅行

人生就像一次旅行。你应该在乎的不是目的地，而是沿途的风景以及看风景的心情。

美食

每个城市都有不同的街景，不同的食物，不同的文化习俗，但是没有你，看起来是一样的。

阅读

独爱于在秋天到来时，那满院盛开的菊花。一个人坐在窗边，品一杯好茶，看一本好书。

音乐

没有那么欢快的声音，铿锵有力，回响悠长，在心里，像一些轻快活泼的精神，在心灵的洗礼。

时尚

不要拘泥于潮流。不要让时尚拥有你，但你决定你是什么，你想通过你的穿着和生活方式表达什么。

电影

我喜欢早上起床的时候一切都未知，我不知道人们会遇到什么，什么样的结局，未知充满了神秘。

香格里拉

五月的香格里拉，是杜鹃花盛开的季节。山脚下，漫山遍野的杜鹃花，汇成一片花的海洋。走进香格里拉，最动人之处是跳傣族舞我爱你，当我们来到闻名的阿诗玛故乡时，傣族男女跳起民族舞欢迎我们。那里的白云、阳光、蓝天、草原、杜鹃花，至今，仍令我梦笔生花。香格里拉的天空纯净静美，好像是佛的眼睛俯视着苍生，怜悯着苍生，所有见过那里的人都会爱上这个自由。

丽江古城

如果说一千个读者心中有一千个哈姆雷特，那么一万个游客心中有一万个丽江也不为过。丽江是喧嚣的，你可以彻夜狂欢；丽江是宁静的，你可以坐在洒满阳光的院落发呆良久。也许是丽江古城内小桥流水人家的柔软风情让人在静谧中左思右想，抑或是夜晚风格各异的酒吧让人魅惑张扬。不过，无论是蓝天白云，雪山潺流，炊烟人家，还是风情酒吧，都如此美好。

桂林

桂林山水没有峨嵋秀、没有青城幽、没有华山险、没有泰山雄、更没有黄山奇，但钱币上桂林的青山绿水活跃在人们的视野里，映衬在人们的脑海里，令人身临其境。桂林，迷蒙的云雾是你的轻纱，绿树红花是你的衣裳，你就像一位降临人间的九天仙女。漓江的水真静啊!静得让你感觉不道它在流动;漓江的水真清啊!清得可以看见江底的沙石;漓江的水真绿啊!绿得仿佛是一块无暇的翡翠。
